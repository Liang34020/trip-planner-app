# Trip Planner - Web Frontend

## 開發指令
```bash
npm run dev        # 啟動開發伺服器
npm run build      # 建置生產版本
npm run preview    # 預覽生產版本
npm run lint       # 執行 ESLint
npm run format     # 執行 Prettier 格式化
```

## 技術棧

- React 18 + TypeScript
- Vite 5
- Tailwind CSS 3
- Zustand（狀態管理）
- React Query（API 請求）
- @dnd-kit（拖曳功能）
